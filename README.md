# A Tiny BASIC Interpreter

This project uses zig 0.15.2

<!-- USAGE -->
## Usage
Download the repository and build as normal by entering the following command in your project root:
```
zig build
```

## Language Reference
[ref](https://en.wikipedia.org/wiki/Tiny_BASIC#Formal_grammar)

<!-- EXAMPLES -->
## Tiny Basic "Hello, World!"
```
PRINT "Hello, World!"
```
More examples can be found in the examples directory.
